# 📚 Ebook Data Collection

Repository ini berisi kumpulan buku (ebook) pilihan yang berfokus pada bidang **Data Analyst, Data Engineering, Data Science, dan Data Storytelling**.

Tujuan dari repository ini adalah untuk membantu siapa saja yang ingin belajar data secara **terstruktur, praktis, dan sesuai kebutuhan industri**.

---

## 🎯 Tujuan Repository

* Menyediakan sumber belajar data dalam satu tempat
* Membantu pemula memahami roadmap belajar data
* Menjadi referensi bagi praktisi untuk memperdalam skill
* Mendukung pembelajaran mandiri (self-learning)

---

Berikut adalah beberapa ebook yang tersedia dalam repository ini:

### 📊 Data Analysis

### 🏛️ Data Governance

### 📖 Data Storytelling

### 🐍 Programming (Python)

### ⚙️ Data Engineering

### 🚀 Product & Analytics

## 🧠 Cara Menggunakan Repository Ini

Jika kamu pemula, disarankan mengikuti urutan berikut:

1. **Mulai dari Data Analysis**

   * Pelajari dasar SQL & Excel

2. **Pelajari Data Storytelling**

   * Cara menyampaikan insight dengan baik

3. **Masuk ke Programming (Python)**

   * Untuk analisis data lebih lanjut

4. **Dalami Data Engineering**

   * Memahami pipeline dan arsitektur data

5. **Pelajari Data Governance & Analytics**

   * Untuk perspektif bisnis & manajemen data
